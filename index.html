1 getElementById

Cara paling dasar untuk mengambil satu elemen HTML berdasarkan atribut id.

Apa itu getElementById?
Bayangkan setiap elemen HTML punya "nama" unik yang disebut id. Dengan document.getElementById("namaId"), kita bisa "mengambil" elemen tersebut supaya bisa kita manipulasi (ubah teks, warna, dll) menggunakan JavaScript.

🔑 Ingat: ID harus unik! Satu halaman hanya boleh punya satu elemen dengan ID yang sama.
JavaScript 
// Langkah 1: Ambil elemen yang punya id="demo-heading"
let heading = document.getElementById("demo-heading");

// Langkah 2: Ubah teksnya
heading.innerText = "Teks ini diubah oleh JavaScript! 🎉";

2 querySelector

Mengambil elemen menggunakan CSS Selector — lebih fleksibel dari getElementById!

querySelector vs querySelectorAll:
• querySelector(".kelas") → Mengambil satu elemen pertama yang cocok
• querySelectorAll(".kelas") → Mengambil semua elemen yang cocok (hasilnya berupa NodeList/daftar)

Bisa pakai selector CSS apa saja: #id, .class, tag, tag.class, dll.
JavaScript 
// querySelector — Ambil elemen pertama dengan class "buah"
let pertama = document.querySelector(".buah");

// querySelectorAll — Ambil SEMUA elemen dengan class "buah"
let semua = document.querySelectorAll(".buah");

// Loop (perulangan) untuk ubah semua elemen
semua.forEach(function(item) {
    item.style.color = "cyan";
});

3 Mengubah Konten

Perbedaan antara innerText, innerHTML, dan textContent.

Tiga cara mengubah konten elemen:
• innerText → Mengubah teks yang terlihat (tanpa tag HTML)
• innerHTML → Mengubah konten HTML (bisa memasukkan tag HTML)
• textContent → Mirip innerText, tapi mengambil SEMUA teks termasuk yang tersembunyi

⚠️ Perhatian: Hati-hati menggunakan innerHTML karena bisa memasukkan kode HTML.
JavaScript 
let kotak = document.getElementById("kotak-konten");

// innerText → hanya teks biasa
kotak.innerText = "Halo Dunia!";

// innerHTML → bisa memasukkan tag HTML
kotak.innerHTML = "<strong>Halo</strong> Dunia!";

4 Mengubah Style (CSS)

Cara mengubah tampilan elemen secara langsung dengan JavaScript.

Cara mengubah CSS via JavaScript:
Gunakan property element.style.namaProperty.

🔑 Penting: Nama CSS yang pakai tanda hubung (seperti background-color) harus ditulis dalam format camelCase → backgroundColor

Contoh lain: font-size → fontSize, border-radius → borderRadius
JavaScript 
let kotak = document.getElementById("style-box");

// Ubah warna background
kotak.style.backgroundColor = "#ff6b6b";

// Ubah ukuran & border
kotak.style.width = "200px";
kotak.style.borderRadius = "50%";

5 Event Listener

Cara membuat elemen merespon aksi user seperti klik, hover, dan ketikan.

Apa itu Event?
Event adalah "kejadian" yang terjadi di halaman web, misalnya:
• click → User mengklik elemen
• mouseover → Mouse masuk ke area elemen
• mouseout → Mouse keluar dari area elemen
• input → User mengetik di form input
• keydown → User menekan tombol keyboard

Kita gunakan addEventListener("namaEvent", fungsi) untuk menambahkan event.
JavaScript 
let tombol = document.getElementById("event-demo-box");

// Tambahkan event listener untuk "click"
tombol.addEventListener("click", function() {
    tombol.style.backgroundColor = "#51cf66";
    tombol.innerText = "Diklik! ✅";
});

6 Membuat & Menghapus Elemen

Cara menambahkan elemen baru ke halaman dan menghapus yang sudah ada.

Langkah membuat elemen baru:
1️⃣ document.createElement("tag") → Buat elemen baru
2️⃣ Set konten/atribut elemen yang baru dibuat
3️⃣ parentElement.appendChild(elemenBaru) → Masukkan ke halaman

Menghapus elemen:
• elemen.remove() → Hapus elemen dari halaman
JavaScript 
// 1. Buat elemen baru
let kartuBaru = document.createElement("span");

// 2. Isi kontennya
kartuBaru.innerText = "🌟 Item Baru";
kartuBaru.className = "item-card";

// 3. Masukkan ke dalam elemen parent
document.getElementById("daftar-item").appendChild(kartuBaru);

// Untuk menghapus:
kartuBaru.remove();
